# Datafuse

## Introduction
Web application that scrapes news articles based on a user query and presents information in a stacked card layout. The goal is to consolidate news information from top ranked sites and provide an excerpt from each article for convenient reading. If the user is interested by the excerpt of the article, they can continue reading the full article by following the provided link. Additional options are being added intermittently, such as choosing an amount of results and filtering news sources based on impartiality.

## Technologies Used
- Backend: Django
- Frontend: React/Redux, HTML/CSS
- Database/Caching: Redis
- Miscellaneous: Google News Search API, BeautifulSoup,
